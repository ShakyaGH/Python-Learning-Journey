# Basic Calculator

A simple calculator project built using Python.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Exponentiation (^)
- Integer Division (//)
- Modulus (%)
- Reset option ($)
- Exit option (X)

## Concepts Used

- Python loops
- Conditional statements
- Lists
- User input handling
- Exception handling (try-except)

## Learning Journey

This is one of my first Python projects while learning programming fundamentals.

I used online resources and AI assistance as a learning support while building this project. The main goal was to understand the logic and practice writing Python code.
